- item06.jpg - 低糖質 フレッシュフルーツタルト
- item07.jpg - カカオ70% リッチチョコレートケーキ
- item08.jpg - カカオ70% プレミアムブラウニー

### ヒーロー画像（/static/images/）

- CS_cake_2_pc.jpg - スライドショー画像2
- CS_cake_3_pc.jpg - スライドショー画像3

### ロゴ
- logo.png - HealSweetsロゴ（コピー済み）

## 画像の追加方法

1. 上記のファイル名で画像を用意してください
2. `/src/main/resources/static/images/` フォルダに配置してください
3. 推奨画像サイズ：
   - 商品画像: 400x400px (正方形)
   - ヒーロー画像: 1920x300px (横長)
   - ロゴ: 高さ60px程度
